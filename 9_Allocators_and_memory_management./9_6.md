# Bit fields
***
```c++
struct S {
    char c;
    char cc;
    int i: 2;
};


int main() {
    std::cout << sizeof(S) << "\n" << // 4
    alignof(S) << '\n';               // 4
}
```

> у битовых полей нельзя брать адрес -> CE
> 
> если битовому полю присвоить значение вне его области значений
> то он просто обрежет (не UB наверно, понять то можно...)
